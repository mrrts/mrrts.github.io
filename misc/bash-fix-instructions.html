El Capitan Terminal Issue

The issue

Mac OSX El Capitan users seeing color codes in git repository directories after installing the supertopher (http://github.com/supertopher) dotfiles:

  • example:

    [\[\e[0;32m\]master\[\e[0m\]]

    instead of

    [master]

The fix

The adjusted .bash_profile from @alexwilliams (http://github.com/Tendoncs)

  1. Go to your home directory:

    cd ~

  2. Clone the fix's repository:

    git clone https://github.com/Tendoncs/BashFix.git

  3. Rename your current .bash_profile to back it up:

    mv .bash_profile .bash_profile_old

  4. Copy the fixed file to your home directory:

    cp BashFix/.bash_profile ./.bash_profile

  5. Reload your bash_profile:

    source ~/.bash_profile
